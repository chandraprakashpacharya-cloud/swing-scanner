⚙️ Setup Guide

1. Install dependencies

pip install flask flask-cors yfinance pandas openpyxl apscheduler numpy

2. Prepare your Excel file

Your Excel file (e.g. NSEV2.xlsx) must have a column named Symbol containing NSE ticker symbols without the .NS suffix — e.g. RELIANCE, INFY, TCS.

3. Run the app

python swing_scanner_app.py --excel NSEV2.xlsx

# Custom port
python swing_scanner_app.py --excel NSEV2.xlsx --port 8080

# Disable auto-scheduler
python swing_scanner_app.py --excel NSEV2.xlsx --no-sched

4. Auto-schedule

The backend automatically runs a scan every weekday at 4:15 PM IST (after NSE close). Results are saved in the results/ folder as JSON + Excel. You can also click Run Scan Now any time.
